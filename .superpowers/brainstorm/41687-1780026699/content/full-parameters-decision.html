📋 第一组：基础核心参数（必填） 3 个参数

#1 querySql 必填

类型: String
默认值: "SELECT * FROM %s"
@EqField: ✅ | @JsonProperty("sqlQuery")

功能描述：

DuckDB SQL 查询语句，用于从内存数据库中查询数据。支持标准 SQL 语法，可包含 JOIN、聚合、窗口函数等。

💡 示例：SELECT o.*, c.name AS customer_name FROM orders o LEFT JOIN customers c ON o.customer_id = c.id

#2 outputTableName 推荐填写

类型: String
默认值: "duckdb_output"
@EqField: ✅

功能描述：

输出表的名称，作为 SQL 查询的目标表名。当有多个源表输入时，系统会为每个源创建独立的临时表。

📝 说明：通常使用默认值即可，除非需要自定义输出表名以便在其他地方引用。

#3 executeQueryOnFullSyncComplete 时序控制

类型: Boolean
默认值: true
@EqField: ✅

功能描述：

控制是否在全量同步完成后立即执行 SQL 查询并发射结果。true=快照模式，false=纯CDC模式。

✅ 推荐 true 时：批量ETL、全量+增量混合
❌ 推荐 false 时：纯实时同步、配合物化视图使用

⚡ 第二组：性能调优参数（可选） 5 个参数

#4 batchSize

类型: Integer
默认值: 1000
范围: 100 - 100000
@EqField: ✅

功能描述：

批处理大小，决定每次写入 DuckDB 的记录数。影响内存占用和 I/O 效率。

⚙️ 调优建议：
• 小数据量（<10万行）：1000-5000
• 中等数据量（10万-100万）：5000-10000
• 大数据量（>100万）：10000-50000

#5 memoryLimitMB

类型: Integer
默认值: 1024 (1GB)
范围: 128 - 65536
@EqField: ✅

功能描述：

DuckDB 内存限制（MB），防止 OOM。超出限制时会触发溢出到磁盘。

#6 queryTimeoutMs

类型: Integer
默认值: 5000 (5秒)
范围: 1000 - 60000
@EqField: ✅

功能描述：

SQL 查询超时时间（毫秒）。防止复杂查询长时间阻塞。

#7 maxActiveSources

类型: Integer
默认值: 50
范围: 1 - 200
@EqField: ✅

功能描述：

最大活跃源数量上限。当超过此数值时，最久未使用的源上下文会被回收。

#8 commitIntervalMs

类型: Integer
默认值: 5000 (5秒)
范围: 1000 - 30000
@EqField: ✅

功能描述：

自动提交间隔（毫秒）。即使批次未满，也会按此间隔强制提交数据。

🛡️ 第三组：错误处理参数（系统级） 2 个参数

#9 errorThresholdCount

类型: Integer
默认值: 100
@EqField: ✅

错误计数阈值。当累计错误数达到此值时触发告警或停止任务。

#10 errorThresholdRate

类型: Double
默认值: 0.01 (1%)
范围: 0.001 - 0.5
@EqField: ✅

错误率阈值。当错误数/总事件数达到此比率时触发告警。

🗄️ 第四组：DuckLake 持久化存储（高级） 3 个参数

#11 duckLakeEnabled 开关

类型: Boolean
默认值: false
@EqField: ✅

功能描述：

启用 DuckLake 持久化存储。开启后数据写入磁盘而非仅内存，支持断点续传。

#12 duckLakeStorageType

类型: String
默认值: "LOCAL"
可选: LOCAL | S3 | GCS | AZURE
@EqField: ✅

功能描述：

存储后端类型。LOCAL=本地文件系统，其他为云存储服务。

#13 duckLakeStoragePath

类型: String
默认值: "/tmp/ducklake"
@EqField: ✅

功能描述：

数据存储路径。本地模式为目录路径，云模式为 Bucket URI。

📊 第五组：实时增量物化视图（专家模式） 4 个独立 + 2 个复合

#14 wideTablePrimaryKey ⭐ 必填

类型: String
默认值: null
示例: "order_id", "user_id"
@EqField: ✅

功能描述：

宽表主键字段名。用于 CDC 事件的 UPDATE/DELETE 匹配，是物化视图的核心标识。

#15 outputChangelogEnabled

类型: Boolean
默认值: false
@EqField: ✅

功能描述：

是否输出宽表 CDC 变更日志。true=持续输出 INSERT/UPDATE/DELETE 事件。

#16 mainTableName + mainTablePrimaryKey

类型: String + String
默认值: null + null
示例: "orders" + "id"
@EqField: ✅

功能描述：

主表配置。定义 JOIN 操作的核心表及其主键，影响 CDC 事件优先级。

#17 fromTables (List) + customJoinQueries (Map)

类型: List<FromTableConfig> + Map<String,String>
默认值: [] + {}
@EqField: ✅

功能描述：

从表列表和自定义 JOIN 查询模板。支持多表关联和复杂的 SQL 定制。
