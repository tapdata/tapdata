🎯 参数分类建议（遵循 Tapdata 规范）

A

基础配置（必填）- 放在节点上

包含参数：

  • querySql - SQL 查询（核心功能）
  • outputTableName - 输出表名
  • executeQueryOnFullSyncComplete - 执行时机

✅ 用户每次使用必须配置

B

性能调优（高级）- 可选放在节点或全局

包含参数：

  • batchSize - 批大小（影响内存）
  • queryTimeoutMs - 查询超时
  • memoryLimitMB - 内存限制
  • maxActiveSources - 最大源数
  • commitIntervalMs - 提交间隔

⚠️ 建议放入「高级设置」Tab

C

错误处理（系统）- 建议全局配置

包含参数：

  • errorThresholdCount - 错误计数阈值
  • errorThresholdRate - 错误率阈值

🔧 通常由管理员统一配置

D

高级功能（专家模式）- 独立 Tab 或条件显示

DuckLake 持久化：

  • duckLakeEnabled
  • duckLakeStorageType
  • duckLakeStoragePath

实时增量物化视图：

  • wideTablePrimaryKey
  • outputChangelogEnabled
  • mainTableName, fromTables
  • customJoinQueries

🚀 仅在启用时显示
